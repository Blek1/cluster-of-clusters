Note that i dont want my computer to just stay without memory/crash cause of memory -> not be able to reclaim it cause the docker container restarts so had to change my settings there:
- `docker stats` = shows me the current containers
- `docker inspect kwok-kwok-etcd | grep RestartPolicy -A 3` 
- ```
"RestartPolicy"{
    "Name": "unless-stopped", 
    "MaximumRetryCount": 0 
    },
```
- 
```
docker update --restart=no kwok-kwok-etcd
docker update --restart=no kwok-kwok-kube-apiserver
