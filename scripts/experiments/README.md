# experiments

Miscellaneous experiments and prototypes. These are assorted attempts, scratchpads, and one-off configurations that are not part of the main workflows in other `scripts/` subdirectories.

## Contents

| Directory | Description |
|---|---|
| `misc/karmada_kind/` | Reproducible Karmada-on-Kind workspace with pinned upstream and patches |
| `misc/karmada_observ/` | Karmada observability stack (Prometheus + Grafana dashboards, RBAC) |
| `misc/kwok-bench/` | Empty — placeholder for future work |
| `misc/kwok-test-two/` | Early KWOK prototype (superseded by `scripts/kwok-testing/`) |
| `misc/simple_docker/` | Standalone Docker Compose stack with Prometheus + Grafana + Nginx |
| `misc/simple_karmada/` | Simplified Karmada setup on Kind |
| `misc/topology/` | Topology configs for small / medium / large scale |

See individual `README.md` files inside each subdirectory for details.
